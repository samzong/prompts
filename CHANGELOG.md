## [0.1.0] - 2025-06-05

### Added

- 初始版本发布
- 系统级提示词管理功能
- 快速搜索和选择界面
- 全局快捷键支持 (Cmd+Shift+P)
- 支持提示词变量替换
- macOS 菜单栏集成
- 支持明暗主题切换
- 提示词分类和标签管理

### Technical

- 基于 Tauri 2.0 + React 18 + TypeScript 构建
- 使用 Zustand 进行状态管理
- Tailwind CSS 样式系统
- 支持多平台构建（macOS, Windows, Linux）
